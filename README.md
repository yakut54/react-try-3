
# react-try-3


## Установка

1. Клонируйте репозиторий с помощью команды:
```bash
git clone git@github.com:yakut54/react-try-3.git
````
2. Установите зависимости с помощью `npm install`

## Использование

### Запуск проекта
```bash
npm start
```

### Сборка для продакшн
```bash
npm run build:prod
```

### Сборка для разработки
```bash
npm run build:dev
```
